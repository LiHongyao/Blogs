



# Windows

## 一、文件 & 目录相关

| Command                    | DESCRIPTION                            |
| -------------------------- | -------------------------------------- |
| `cd path`                  | 打开指定                               |
| `cd ..`                    | 返回上一级目录                         |
| `cd ~`                     | 返回用户目录                           |
| `cd /`                     | 返回根目录                             |
| `dir`                      | 查看目录下的文件                       |
| `mkdir dirName[/fileName]` | 新建目录                               |
| `cd > fileName`            | 新建文件                               |
| `del [-r] fileName`        | 删除文件，`-r`：删除目录及目录下的文件 |
| `car fileName`             | 查看文件内容                           |
| `move A B`                 | 移动文件                               |
