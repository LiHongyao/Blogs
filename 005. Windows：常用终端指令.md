



# Windows

## 一、目录相关

| Command   | DESCRIPTION      |
| --------- | ---------------- |
| `cd path` | 打开指定         |
| `cd ..`   | 返回上一级目录   |
| `cd ~`    | 返回用户目录     |
| `cd /`    | 返回根目录       |
| `dir`     | 查看目录下的文件 |

## 一、文件操作

| Command         | DESCRIPTION  |
| --------------- | ------------ |
| `cd > fileName` | 新建文件     |
| `del fileName`  | 删除文件     |
| `car fileName`  | 查看文件内容 |
